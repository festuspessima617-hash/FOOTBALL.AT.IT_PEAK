# Football At Its Peak

A comprehensive website for Football At Its Peak, an elite football development company specializing in branding, training systems, and player development.

## Features

- **Modern Design**: Dark theme with red accents, fully responsive design
- **Interactive Elements**: Smooth animations, hover effects, and scroll-triggered animations
- **Gallery**: Showcase of football images and moments
- **Application Forms**: Separate forms for players and coaches
- **Contact System**: Integrated contact form with Netlify form handling
- **SEO Optimized**: Meta tags and semantic HTML structure

## Pages

- **Home** (`index.html`): Landing page with hero section, features, and statistics
- **About** (`about.html`): Company mission, services overview, and team information
- **Services** (`services.html`): Detailed service offerings
- **Gallery** (`gallery.html`): Image gallery using local football images
- **Apply** (`apply.html`): Application forms for players and coaches
- **Contact** (`contact.html`): Contact information and inquiry form
- **Success** (`success.html`): Form submission confirmation page

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript**: Interactive features including scroll animations, form validation, and gallery lightbox
- **Netlify Forms**: Serverless form handling for applications and contact

## Setup

1. Clone or download the project files
2. Ensure all files are in the same directory structure
3. For local development, use a local server (e.g., `python -m http.server`)
4. For production, deploy to Netlify for form handling

## Customization

- **Colors**: Main colors are defined in `style.css` (dark background, red accents)
- **Content**: Update text content in respective HTML files
- **Images**: Replace images in the `images/` folder
- **Forms**: Modify form fields in HTML files and update Netlify form names accordingly
